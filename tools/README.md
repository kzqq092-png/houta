# 工具目录

本目录包含项目的各种工具和脚本。

## 工具列表

### 系统检查工具
- `comprehensive_pattern_system_check.py` - 全面的形态识别系统检查工具
- `comprehensive_strategy_system_test.py` - 策略系统全面测试工具
- `comprehensive_unified_backtest_test.py` - 统一回测引擎测试工具

## 使用说明

这些工具可以独立运行，用于系统维护、测试和诊断。

### 运行方式
```bash
# 从项目根目录运行
python tools/comprehensive_pattern_system_check.py
```

## 注意事项

- 这些工具主要用于开发和维护阶段
- 运行前请确保数据库和相关依赖已正确配置
- 工具会生成详细的报告文件
