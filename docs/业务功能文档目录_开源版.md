# HIkyuu-UI 业务功能文档目录 - 开源版

## 项目概述

HIkyuu-UI 是一个基于 WebGPU 硬件加速的量化交易分析系统，提供股票数据分析、AI模型预测、策略开发、插件扩展等核心功能。本目录按照业务框架对所有文档进行分类，为开源用户提供清晰的功能导航。
