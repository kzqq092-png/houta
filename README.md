# FactorWeave-Quant 2.0

## 项目简介

FactorWeave-Quant是一个基于Python的量化交易系统，集成了数据获取、技术分析、策略回测、实时交易等功能。

## 主要特性

### 🚀 核心功能
- **多数据源支持**: 支持新浪、东方财富、通达信、AKShare等多种数据源
- **智能插件系统**: TET框架插件架构，支持动态加载和配置
- **高性能计算**: WebGPU硬件加速，多线程并行处理
- **实时数据**: 支持实时行情数据获取和处理
- **策略回测**: 完整的回测引擎和性能分析
- **风险控制**: 多层级风险管理和监控系统

### 🎯 技术架构
- **统一数据管理**: UnifiedDataManager统一数据访问接口
- **服务化架构**: 基于服务容器的模块化设计
- **缓存系统**: 多级缓存提升数据访问性能
- **事件驱动**: 异步事件总线支持实时响应
- **AI集成**: 深度学习模型支持智能预测

### 📊 数据支持
- **股票数据**: A股、港股、美股实时和历史数据
- **指数数据**: 各类指数和ETF数据
- **基本面数据**: 财务数据、公司信息等
- **宏观数据**: 经济指标、行业数据等
- **情绪数据**: 市场情绪分析和舆情监控

## 快速开始

### 环境要求
- Python 3.8+
- PyQt5/PySide2
- 支持WebGPU的显卡（可选）

### 安装依赖
```bash
pip install -r requirements.txt
```

### 启动系统
```bash
python main.py
```

## 系统架构

### 核心模块
- `core/`: 核心业务逻辑
- `gui/`: 用户界面组件
- `plugins/`: 插件系统
- `db/`: 数据库管理
- `utils/`: 工具函数

### 数据流
1. 数据源插件 → TET路由器 → 统一数据管理器
2. 数据处理 → 缓存系统 → 业务服务
3. 用户界面 → 事件总线 → 后端服务

## 开发指南

### 插件开发
系统支持多种类型的插件：
- 数据源插件（IDataSourcePlugin）
- 指标插件（IIndicatorPlugin）
- 策略插件（IStrategyPlugin）

### 服务扩展
基于AsyncBaseService可以轻松扩展新的业务服务。
