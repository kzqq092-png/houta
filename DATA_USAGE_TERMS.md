# YS-Quant‌ 数据使用条款

## 📋 数据使用声明

### 1. 适用范围
本使用条款适用于YS-Quant‌量化交易系统中所有数据源的使用，包括但不限于：
- 本地数据源（HikyuuDataSource）
- 开源数据源（AkshareDataSource）  
- 公开API数据源（东方财富、新浪财经、同花顺等）
- 第三方插件数据源（Yahoo Finance等）

### 2. 使用目的限制

#### ✅ 允许的使用目的
- **个人投资研究**：用于个人投资决策分析和研究
- **学术研究**：用于金融量化相关的学术研究
- **教育学习**：用于量化交易技术的学习和教育
- **非商业分析**：用于非营利性的数据分析和回测

#### ❌ 禁止的使用目的
- **商业转售**：禁止将获取的数据进行商业转售
- **大规模爬取**：禁止进行大规模、高频率的数据爬取
- **恶意使用**：禁止用于任何可能损害数据提供方利益的行为
- **版权侵犯**：禁止侵犯数据提供方的知识产权

### 3. 数据来源说明

#### 本地数据源
- **数据来源**：HIkyuu量化交易框架本地数据库
- **使用条款**：遵循HIkyuu项目的开源许可证
- **合规性**：完全合规，无版权风险

#### 开源数据源（AkShare）
- **数据来源**：AkShare开源金融数据接口
- **使用条款**：遵循AkShare项目的BSD-3-Clause许可证
- **合规性**：开源项目，使用合规
- **项目地址**：https://github.com/akfamily/akshare

#### 公开API数据源
- **东方财富**：
  - 数据来源：东方财富网公开API接口
  - 使用限制：仅限个人研究使用，请遵守网站使用条款
  - 请求频率：建议不超过每分钟60次请求

- **新浪财经**：
  - 数据来源：新浪财经公开数据接口
  - 使用限制：仅限个人研究使用，请遵守网站使用条款
  - 请求频率：建议不超过每分钟60次请求

- **同花顺**：
  - 数据来源：同花顺公开数据接口
  - 使用限制：仅限个人研究使用，请遵守网站使用条款
  - 请求频率：建议不超过每分钟60次请求

### 5. 数据质量保证

#### 数据质量检查
YS-Quant‌提供内置的数据质量检查功能：
- **完整性检查**：检查数据是否完整，无缺失值
- **准确性检查**：检查价格数据的逻辑合理性
- **一致性检查**：检查数据格式和结构的一致性
- **时效性检查**：检查数据的更新时间和时效性

#### 质量问题处理
- **数据异常报告**：系统会自动识别和报告数据异常
- **质量评分**：为每个数据源提供质量评分
- **问题追踪**：记录和追踪数据质量问题
- **改进建议**：提供数据质量改进建议

### 6. 隐私保护

#### 用户隐私
- **数据本地化**：用户数据主要存储在本地，保护用户隐私
- **最小化收集**：仅收集必要的数据，不收集个人敏感信息
- **加密存储**：重要数据采用加密存储
- **访问控制**：实施严格的数据访问控制

#### 第三方数据
- **匿名化处理**：对第三方数据进行匿名化处理
- **最小化传输**：仅传输必要的数据字段
- **安全传输**：使用HTTPS等安全协议传输数据

### 7. 技术规范

#### 请求频率限制
```python
# 推荐的请求频率设置
RATE_LIMITS = {
    'eastmoney': 60,    # 每分钟60次
    'sina': 60,         # 每分钟60次
    'tonghuashun': 60,  # 每分钟60次
    'akshare': 120,     # 每分钟120次（开源，相对宽松）
    'hikyuu': -1,       # 无限制（本地数据）
}
```

#### 错误处理规范
```python
# 推荐的错误处理方式
try:
    data = data_source.get_data(symbol)
except RateLimitError:
    # 等待并重试
    time.sleep(60)
    data = data_source.get_data(symbol)
except NetworkError:
    # 网络错误，使用缓存数据
    data = cache.get(symbol)
except DataQualityError:
    # 数据质量问题，记录并报告
    logger.warning(f"Data quality issue for {symbol}")
    data = None
```

#### 缓存策略
```python
# 推荐的缓存设置
CACHE_SETTINGS = {
    'real_time_data': 30,     # 实时数据缓存30秒
    'daily_data': 3600,       # 日线数据缓存1小时
    'historical_data': 86400, # 历史数据缓存24小时
}
```
